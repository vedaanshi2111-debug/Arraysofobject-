Array of Objects in C++
Overview

This program demonstrates how to create and use an array of objects in C++.
Each object represents a student with attributes like name and age.
The program allows users to input and display details for multiple students.

Sample Output

--- Enter student details ---

Student 1:
Enter name: Riya
Enter age: 20

Student 2:
Enter name: Aman
Enter age: 21

Student 3:
Enter name: Vedaanshi
Enter age: 22

Name: Riya, Age: 20
Name: Aman, Age: 21
